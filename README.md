## ***segundo challenge telecomx***
**Descripción del proyecto:**
En este proyecto se realiza un análisis del comportamiento de los clientes de nuestra empresa TelecomX y desarrollar modelos de machine learning que nos ayudan a predecir la evasión de servicio. Esta cancelación representa un gran problema para nuestra organización, se busca proponer estrategias.

**Tecnologías y librerías usadas**:
- Pandas: carga, normalización y limpieza de datos
- Matplotlib: Visualizaciones base
- Seaborn: Gráficas estadísticas
- NumPy
- Scikit-learn
- Imbalanced-learn
- Google Colab

**Flujo de trabajo**:
1. Extracción de datos
2. Exploración inicial
3. Limpieza de datos
4. Análisis exploratorio
5. Transformación de variables categóricas
6. División del dataset en entrenamiento y prueba
7. Balanceo con SMOTE
8. Entrenamiento de modelos
9. Evaluación
10. Selección del mejor modelo
11. Descarga del modelo
12. Informe final

**Modelos utilizados**
- Regresión Logística **Modelo champion**
- Random Forest
- Árbol de Decisión

**Métricas de evaluación**
- Accuracy: proporción de predicciones correctas.
- Precision: proporción de predicciones positivas correctas.
- Recall: capacidad del modelo para identificar correctamente los casos de cancelación.
- F1 Score: balance entre precisión y recall.
- Matriz de confusión: análisis de errores de clasificación.

**Archivos**

- modelo_champion.pkl → Modelo final entrenado
